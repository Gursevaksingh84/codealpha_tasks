# Student Grade Tracker (Java)

A simple **console-based Java application** to manage and analyze student grades.
This program allows users to input student names and their grades, store them using an **ArrayList**, and generate a summary report including the **average, highest, and lowest scores**.

## Features

* Input multiple student names and grades
* Store data using Java **ArrayList**
* Calculate:

  * Average score
  * Highest score
  * Lowest score
* Display a **summary report of all students**
* Simple **console-based interface**

## Technologies Used

* Java
* ArrayList
* Scanner (for user input)

## How to Run

1. Clone the repository or download the project.

2. Compile the Java file:

```
javac StudentGradeTracker.java
```

3. Run the program:

```
java StudentGradeTracker
```
